# enigma

Implementation of a virtual Enigma machine in various languages

## Implementation limitations

* No switchboard
* Only one type of rotor
* Only full tick rotors
* Upper case ASCII leters only (65 - 90)

### Rotor implementation validation

> Rotor position *C*

A > C > M > K

K > M > C > A

D > F > G > E

E > F > G > D
